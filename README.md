# smartcamer
简单的ai微信小程序，后台调用百度接口。<br>
包含功能：文字识别、通用物品识别、动物识别、花草识别、身份证识别、汽车识别和翻译。<br>
选中的图片可以进行裁剪，使用的是image-cropper插件。<br>
